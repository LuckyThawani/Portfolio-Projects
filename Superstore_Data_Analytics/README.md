# 📊 Superstore Analytics Project

## 📌 Overview
This project analyzes a retail Superstore dataset to uncover insights into sales, profit, discount impact, and customer behavior.

## 🎯 Problem Statement
The objective is to identify key factors affecting profitability and provide actionable insights to improve business performance.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn (for clustering)

## 📊 Key Analysis Performed
- Exploratory Data Analysis (EDA)
- Handling missing values
- Profit vs Discount analysis
- City-wise profit/loss analysis
- K-Means clustering for customer/segment insights
- Statistical testing (A/B testing, p-value interpretation)

## 📈 Key Insights
- High discounts negatively impact profit
- Certain cities consistently generate losses
- Customer segments show distinct purchasing patterns
- Outliers significantly affect model performance
